# 206. 反转链表
```
反转一个单链表。
```

## 解法1
```
迭代的方式
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func reverseList(head *ListNode) *ListNode {
    var newHeade *ListNode = nil
    // 每次遍历一个节点,同时改变节点的指向
    for head != nil {
        next := head.Next
        head.Next = newHeade
        newHeade = head
        head = next
    }
    return newHeade
}
```


# 24. 两两交换链表中的节点
```
给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。
你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。
```

## 解法
```
思路：递归的方式
1.确定递归的终止条件(当前节点和下一个节点都存在)
2.找到重复的子问题(每次交换2个节点的指针)
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func swapPairs(head *ListNode) *ListNode {
    // 终止条件
    if head == nil || head.Next == nil {
        return head
    }

    // 子操作
    tmp := head.Next
    next := tmp.Next
    tmp.Next = head
    head.Next = swapPairs(next)
    return tmp  // 每次把第二个节点返回
}
```

# 141. 环形链表
```
给定一个链表，判断链表中是否有环。
为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。
```

## 解法1
```
借助map，如果有环会在map中找到
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func hasCycle(head *ListNode) bool {
    // 借助map,每次走过的存入map,如果有环会在map中找到(说明是重复的)
    // 无环的每个节点map中都会是唯一的
    passNode := make(map[*ListNode]bool, 0)
    for head != nil {
        if _, exists := passNode[head]; exists {
            return true
        }
        passNode[head] = true
        head = head.Next
    }
    return false
}
```

## 解法2
```
把走过的节点设置为一个不会在链表中出现的值
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func hasCycle(head *ListNode) bool {    // 走自己的路让别人无路可走思路
    for head != nil {
        if head.Val == 123456789 {       // 这是自己走过的路，说明有环
            return true
        }
        head.Val = 123456789
        head = head.Next
    }
    return false
}
```

## 解法3
```
龟兔赛跑，看看是否能够相遇到
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func hasCycle(head *ListNode) bool {
    if head == nil {
        return false
    }

    // 快慢指针-位于同样的起点(龟兔赛跑)
    slower, faster := head, head
    for faster != nil && faster.Next != nil {   // 走到头了还没相遇
        faster = faster.Next.Next   // 兔子走两步
        slower = slower.Next        // 乌龟走一步
        // 是否相遇到
        if faster == slower {
            return true
        }
    }
    return false
}
```

# 142. 环形链表 II
```
给定一个链表，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。
为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。
说明：不允许修改给定的链表。
```

## 解法1
```
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func detectCycle(head *ListNode) *ListNode {
    // 借助map
    mapNode := make(map[*ListNode]bool, 0)
    for head != nil {
        if _, found := mapNode[head]; found {
            return head
        }
        mapNode[head] = true
        head = head.Next
    }
    return nil
}
```

## 解法2
```
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func detectCycle(head *ListNode) *ListNode {
    // 找到第一次相遇的点,然后把fast放到head处同时走一步相遇即是入口点
    isOnceMeet := false
    slow, fast := head, head    // 先从同一个起跑线开始
    for fast != nil && fast.Next != nil {
        slow = slow.Next
        fast = fast.Next.Next
        if slow == fast {
            isOnceMeet = true
            fast = head
            break
        }
    }

    // 第一次相遇都没有
    if !isOnceMeet {
        return nil
    }

    // 找到第一次相遇的点,把fast放到开始位置,同时走一步
    for fast != slow {
        fast = fast.Next
        slow = slow.Next
    }
    return slow
}
```
