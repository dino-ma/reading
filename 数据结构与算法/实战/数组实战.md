# 283. 移动零
```
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。
说明:
    必须在原数组上操作，不能拷贝额外的数组。
    尽量减少操作次数。
```

## 解法1
```
func moveZeroes(nums []int)  {
    // 分2步,先把非0的操作,最后补0
    total := len(nums)
    notZeroNum := 0
    for i := 0; i < total; i++ {
        if nums[i] != 0 {
            nums[notZeroNum] = nums[i]
            notZeroNum++
        }
    }

    for notZeroNum < total {
        nums[notZeroNum] = 0
        notZeroNum++
    }
}
```

## 解法2
```
思路：1个循环，把非0和0的位置进行交换，滚雪球方式(交换0和非0的元素的位置)，对0的个数计数，把非0的和最左边的0交换位置(跳到长度的最左边即可)
func moveZeroes(nums []int)  {
    // 统计0的个数,把非0的和0进行位置交换
    total := len(nums)
    zeroNum := 0
    for i := 0; i < total; i++ {
        if nums[i] == 0 {
            zeroNum++
        } else if(zeroNum > 0) {
            nums[i-zeroNum] = nums[i]
            nums[i] = 0
        }
    }
}
```


# 11.盛最多水的容器
```
给你 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。
```

## 解法1
```
思路：暴力破解，2层循环找出最大值
func maxArea(height []int) int {
    // 面积=长x宽,暴力求解
    total := len(height)
    maxArea := 0
    for i := 0; i < total - 1; i++ {    // 第一个点的下标
        for j := i + 1; j < total; j ++ { // 第二个点的下标
            minHeight := height[i]
            if minHeight > height[j] {
                minHeight = height[j]
            }

            area := (j - i) * minHeight
            if area > maxArea {
                maxArea = area
            }
        }
    }
    return maxArea
}
```

## 解法2
```
思路：一层循环,使用2个指针往两头开始夹逼,不断的找最高的height
func maxArea(height []int) int {
    // 使用2个指针往里夹逼方式
    total := len(height)
    maxArea := 0
    left, right := 0, total - 1
    for left < right {
        isMoveLeft := true
        minHeight := height[left]
        if minHeight > height[right] {
            minHeight = height[right]
            isMoveLeft = false
        }

        area := (right - left) * minHeight
        if area > maxArea {
            maxArea = area
        }

        // 移动指针
        if isMoveLeft {
            left++
        } else {
            right--
        }
    }
    return maxArea
}
```

# 70.爬楼梯
```
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？
注意：给定 n 是一个正整数。
```

## 解法1
```
思路：归纳法得出递归公式，使用map把每个台阶的走法数都存储起来
func climbStairs(n int) int {
    // 通过归纳得 f(n) = f(n-1) + f(n-2),做好边界的处理
    // 借助map,把每种阶梯的走法都记录下来
    mapNums := make(map[int]int, 0)
    mapNums[0],mapNums[1],mapNums[2] = 0,1,2
    for i := 3;  i <= n; i++ {
        mapNums[i] = mapNums[i-1] + mapNums[i-2]
    }
    return mapNums[n]
}
```

## 解法2
```
思路：使用map会消耗较多的空间，能否改为只使用变量去存储结果
func climbStairs(n int) int {
    // 通过归纳得 f(n) = f(n-1) + f(n-2),做好边界的处理
    // 借助变量把结果值存储即可
    if n <= 3 {
        return n
    }
    stepOne, stepTwo, rtn := 1, 2, 0
    for i := 3;  i <= n; i++ {
        rtn = stepTwo + stepOne
        stepOne = stepTwo
        stepTwo = rtn
    }
    return rtn
}
```

# 1.两数之和
```
给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。
你可以假设每种输入只会对应一个答案。但是，数组中同一个元素不能使用两遍。
```

## 解法1
```
思路：暴力求解，两重循环去找到那2个数
func twoSum(nums []int, target int) []int {
    // 暴力求解
    rtn := make([]int, 0)
    total := len(nums)
    for i := 0; i < total - 1; i++ {
        for j := i + 1; j < total; j++ {
            if nums[i] + nums[j] == target {
                rtn = append(rtn, i, j)
            }
        }
    }
    return rtn
}
```

## 解法2
```
思路：使用一个hashmap，用空间换时间的方式
func twoSum(nums []int, target int) []int {
    // 1、第一次遍历使用hashmap把数据进行存储，把值作为key，需要查找的下标做为val
    // 2、第二次遍历查找hashmap即可
    total := len(nums)
    hashMap := make(map[int]int, 0)
    for i := 0; i < total; i++ {
        key := nums[i]
        hashMap[key] = i
    }

    rtn := make([]int, 0) 
    for i := 0; i < total; i++ { 
        findNum := target - nums[i]
        // 同一个元素不能使用2遍
        index, find := hashMap[findNum]
        if find && index != i {
            rtn = append(rtn, i)
            rtn = append(rtn, index)
            return rtn
        }
    }
    return rtn
}
```

## 解法3
```
思路：使用hashmap，遍历一遍即可
func twoSum(nums []int, target int) []int {
    // 一层循环 + 借助一个map
    total := len(nums)
    rtn := make([]int, 0)
    mapV := make(map[int]int, 0) // 值为key,value为index
    for i := 0; i < total; i++ {
        otherval := target - nums[i]
        index, found := mapV[otherval]
        if found {
            rtn = append(rtn, index, i)
        } else {
            mapV[nums[i]] = i
        }
    }
    return rtn
}
```

## 说明
```
这个两数之和，因为是找下标。所以还不能对数组排序，排序后下标就重新分配了
如果是找两数之和的2个元素的值，可以采用排序+两头夹逼方式
```

# 15.三数之和
```
给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。
注意：答案中不可以包含重复的三元组。
```

## 解法
```
思路：排序 + 指针往里夹逼
func threeSum(nums []int) [][]int {
    // 进行排序
    sort.Ints(nums)
    rtn := make([][]int, 0)
    total := len(nums)
    for i := 0; i < total - 2; i++ {
        // 第一个数 || 第二个数不等于排序好的前一个数
        if i > 0 && nums[i] == nums[i-1] {
            continue
        }

        // 使用指针往里夹逼
        left, right := i+1, total -1
        for left < right {
            count := nums[left] + nums[right] + nums[i]
            if count == 0 {
                tmp := make([]int, 0)
                tmp = append(tmp, nums[left], nums[right], nums[i])
                rtn = append(rtn, tmp)
                // 移动指针继续找
                left++ 
                right--

                // 去掉使用过的元素(当前找到的元素和之前的相同,需要去掉)
                for left < right && nums[left] == nums[left-1] {
                    left++        
                }
                for left < right && nums[right] == nums[right+1] {
                    right--
                }
            } else if count > 0 {
                right--
            } else if count < 0 {
                left++
            }
        }
    }
    return rtn
}
```
