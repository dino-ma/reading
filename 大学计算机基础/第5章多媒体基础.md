## 媒体的分类
```
感觉媒体
    直接作用于人的感官，使人能直接产生感觉的一类媒体
    人们视觉、听觉、触觉、嗅觉和味觉感觉到的各种客观存在
    听觉：说话、大自然的声音，嗅觉：问到的气味，视觉：看到的图像，视觉、听觉：视频
    计算机领域的媒体主要指的是感觉媒体（文本、图形、图像、声音、视频）
表示媒体
    人们利用计算机加工、处理和传输感觉媒体而认为构造出来的一种媒体（感觉媒体数字化以后的表示形式）
显示媒体
    表示媒体传输中的电信号与感觉媒体之间转换所使用的一类硬件设备。输入设备和输出设备
存储媒体
    用来存储表示媒体的设备
传输媒体
    将媒体从一处传送到另一处的物理载体。电话线、光纤、空气等等
```

## 多媒体计算机
```
具有多种媒体功能的个人计算机
可以对多种媒体信息进行综合处理，
不仅仅处理文本，还能处理图形、图像、声音、视频等等

硬件系统：在个人计算机的基础上，增加多媒体输入和输出设备及其接口卡。
        声卡、显卡、视频卡、高速USB接口

软件系统：包括多媒体操作系统、支持多媒体数据开发的应用工具软件和多媒体应用软件等。

多媒体技术的特点
    集成性
        媒体的集成
            不能只处理单个媒体，能综合处理声音、文本、图像、图形、视频等多种信息
            信息要同步组合，成为一个完整的多媒体信息系统
        设备的集成
            有高速并行的CPU
            有大容量的存储器
            有适合多媒体传输的输入输出通道
            以及相关的外设和通信网络接口
    实时性
        相比文本，声音和视频的数据量很大。但是需要保持这些信息的同步传输
    交互性
        人机交互需要通过人机界面，人机界面友好才能让用户用起来得心应手
    多样性
        信息媒体的多样性
        媒体处理方式的多样式
```

## 彩色空间和颜色模型
```
颜色的形成：在特定光源照射下所反射的一些可见光谱成分，在眼睛所引起的一些视觉效果。
色调：色调反应色彩的感觉，指的颜色的种类，它和光波的波长有关。不同的光波反映不同的颜色感。
饱和度：反映的是颜色的深浅或浓淡，用掺入白光的程度来衡量。掺的白光越多饱和度越低。
亮度：它反映颜色作用于人眼所引起的明暗视觉程度。

基色：指的是互为独立的单色，任何基色不能由其它颜色混合产生，也就是它不是其它颜色合成的
三基色原理：红、绿、蓝（简称RGB）这三种颜色是相互独立的，用这3种颜色按照不同比例混合就能调配出自然界中丰富多彩的颜色。

RGB颜色空间：相加混合颜色空间、CMY颜色空间：相减混合颜色空间
```

## 图形和图像
```
对空间连续坐标(x,y)离散化就是采样
图像函数f(x,y)颜色的离散化就是量化
两种离散化结合在一起叫做数字化
离散化以后也就是进行数字化以后得到的图像称为数字图像

图形
    是由计算机绘制而成的几何图、矢量图
    像AutoCAD、CoolDraw等软件绘制的图形

图像
    是有颜色的像素点集合
    一般是由扫描仪、数字照相机、摄像机等输入设备捕捉的真实场景

区分图形和图像
    1、概念上
        图形是矢量的概念，基本元素是图元，也就是图形指令
        图像是位图的概念，基本元素是像素，图像比图行更逼真
    2、存储方式
        图形存储的是画图的函数，图形的颜色数目与文件的大小无关
        图像存储的是像素的颜色信息以及灰度信息，图像色数越多，所占二进制位数越多，容量越大。
    3、处理方式
        对图形可以旋转、扭曲、拉伸
        对图像可以进行对比度增强、边缘检测等等
    4、与分辨率的关系
        矢量图与分辨率没有关系，因此可以将它压缩放到任意大小和任意分辨率在输出设备上打印出来，放大不失真
        位图图像与分辨率有关，整个图像是由很多像素组合而成的，放大到一定程度会失真，出现马赛克现象
    5、存储的格式
        常见的矢量图形文件存储格式有：WMF、DRW、CDR、DXF、EPS等
        常见的位图图像文件存储格式有：BMP、TIF、GIF、JPG、PNG等

图像的属性
    分辨率  
        显示分辨率：显示屏幕上显示出来的像素
        图像分辨率：一副图像的像素密度
        打印分辨率：打印机输出图像时采用的分辨率
    颜色深度
        存储每个像素所使用到的二进制位数
        未经压缩的图像大小(字节) = 图像分辨率 X 颜色深度 / 8

图形文件格式
    SVG格式：可缩放的矢量图形格式，由W3C联盟进行开发
    WMF格式：是微软公司定义的一种Windows平台下的图元文件格式
    AI文件：是Adobe公司的lllustrator软件的输出格式
    DXF文件：是AutoCAD中的矢量文件格式，它以ASCII码方式存储
    SWF文件：是二维动画软件FLASH中的矢量动画格式

图像文件格式
    BMP格式
    GIF格式：增加了渐显方式，在图像传输过程中，用户可以先看到图像的大致轮廓，然后随着传输过程的继续而逐步看清图像中的细节部分
    JPEG格式
    TIFF格式
    PNG格式：设计的目的是试图替代GIF和TIFF文件格式，同时增加一些GIF文件格式所不具备的特性
```

## 图像的压缩编码技术
```
无损压缩：图像数据中很多重复数据，使用数学方法表示这些重复的数据就可以减少数据量
有损压缩：人的眼睛对图像颜色和细节的辨认有一个极限，把超过极限的部分去掉，这也达到了压缩数据的目的
静态图像压缩标准JPEG：JPEG标准所使用的算法叫”多灰度连续色调静止图像压缩编码“

JPEG200:进一步改进JPEG压缩算法的性能，以适应低带宽、高噪音的环境
    使用了小波变换为主的多解析编码方式，压缩比在JPEG基础上再提高10%~30%，压缩后的图像更加细腻、平滑
    提高有损和无损压缩，像医学领域、档案保存的时候是不能进行有损压缩的
    实现渐进传输(逐步显示)、提供感兴趣的区域压缩、码流的随机访问和处理
```

## 音频的基本特性和数字化
```
音频信号的主要物理参数
    频率：每秒震动的次数，以HZ为单位
    周期：两个相邻波之间的时间长度，频率和周期互为倒数
    振幅：波的高低幅度，表示声音的强弱

声音的三要素
    音调：声音的基频，频率越高，音调越高
    音色：由声音的频谱决定的
    响度：也就是音量

模拟声音的数字化过程主要是通过采用、量化、编码，将模拟声音变成数字信号    
```

## 音频文件格式和压缩技术
```
WAV格式：真实声音的数字化文件，数据量很大。把模拟音频信号经过采用和量化后直接存到文件
CD格式：CD格式是CD唱片采用的格式，记录的是波形流，音质好、高保真。确定是无法编辑、文件很大
MP3格式：MPEG音频压缩标准，也是最流行的声音文件格式，压缩比高，在网络音频通信方面应用广泛
RealAudio格式：流媒体音频文件，有三类文件后缀.ra、.rm和.ram
WMA格式：Windows media Audio文件格式
MIDI格式：音乐工业的数据通信标准

无损压缩
有损压缩
```
