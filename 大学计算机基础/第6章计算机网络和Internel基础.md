# 计算机网络概述
```
将具有独立功能的多台计算机及其外部设备，通过通信设备和线路连接起来，
在网络软件的管理和协调下，实现资源共享和数据通信的计算机系统。

节点指的是”具有独立功能的多台计算机及其外部设备“
节点与节点之间通过通信设备和线路进行连接，只有连接起来才能成网
节点需要再网络软件的管理和协调下，进行信息交互，信息交互的内容主要包含资源共享和数据通信

数据通信、资源共享、分布式处理
```

## 计算机网络分类
```
网络的分布范围
    局域网LAN：10KM以内
    广域网WAN：几十-几千公里
    城域网MAN：覆盖一个城市或地区，几十-上百公里

网络的拓扑结构
    指网络中各个节点的相互位置和它们相互连接成的几何图形

    星型结构：有一个中心节点，其他节点和中心节点进行连接，
            其它节点之间不能直接通向，需要经过中心节点的控制进行通信
            大部分的局域网采用星型结构
            优点：结构简单、组网容易，网络延迟时间短，误码率低，费用也低，便于维护和管理
            缺点：共享能力较差，通信线路利用率不高，中心节点负担过重

    总线型结构：用一条称为总线的中央电缆，将各个节点连接起来。
              总线型拓扑结构是一种共享通路的物理结构。
              这种结构中总线具有信息的双向传输功能，普遍用于局域网的连接
              优点：具有结构简单灵活、共享资源能力强、可靠性高
              缺点：总线长度有一定限制，一条总线也只能连接一定数量的结点，并且总线自身的故障会导致系统的崩溃

    环型结构：所有节点连接成一个闭合的环型通信线路。环路上任何结点均可以请求发送信息。
            请求一旦被批准，便可以向环路发送信息。环型拓扑主要用于主干网络中
            优点：可靠性较高、实时性强、安装容易，费用较低，电缆故障容易查找和排除
            缺点：结点过多的时候，传输效率低、扩充不方便。

    树型结构、网状结构：以前三种拓扑结构为基础的混合型
        树形结构是总线型结构的一个扩展，它就像一棵”根“朝上的树。
        这种拓扑结构的网络一般用于军事单位、政府部门等，上下界限相当严格和层次分明的部门。
        优点：容易扩展、可靠性强
        缺点：专用性强

        网状结构主要用于广域网，将多个子网连接起来，构成网际拓扑结构。
        优点：由于结点之间有多条线路相连，可靠性高、资源共享方便、通信效率高
        缺点：结构复杂、建设成本较高

网络的传输介质
    有线网、无线网

网络的通信速率、通信协议
    低速网、中速网、高速网

网络的使用用户
    公用网、专用网
```

## 计算机网络的传输介质和通信协议
```
有线介质
    光缆(光纤)、铜缆(双绞线、同轴电缆)
    双绞线
        屏蔽双绞线
        非屏蔽双绞线

无线介质
    无线电波、蓝牙、红外

网络性能的指标
    带宽：网络通信线路所能传送数据的能力
    误码率：出错率，指的是传输出错的比特数占总的比特数的比例
    时延：将一个比特从一个地方传输到另一个地方所花费的时间

通信协议
    节点和节点之间想要真正实现通信交互，还需要网络软件的管理和协调
    通信的双方必须遵循共同的规则和约定，协议由国际化标准组织ISO、电子和电气工程师协会IEEE、Internet工程小组IETF来制定
```

## 局域网简介
```
以太网也叫有线局域网，是当前最普遍的局域网技术。
以太网的传输是广播式传输，广播方式会出现的问题
    信息碰撞、广播风暴、模糊不清
以太网采用载波监听多路访问及冲突检测CSMA/CD方法来解决这种”冲突问题“

局域网的组建
    计算机+通信线路(通信介质)+计算机接口(网卡)+交换机

无线局域网的组建
    计算机+网卡(具备无线技术的无线网卡)+AP(无线接入点，用来扩展无线网络)
```

## Internet的诞生和发展
```
1962年：世界处于美苏冷战期间，美国军方为了保证自己的军事指挥系统在受到前苏联打击后，不至于全盘崩溃
        认为有必要设计出一种分散的指挥系统
1969年：美国国防部主持建设了一个名为阿帕网的网络
1974年：TCP/IP协议，解决了不同网络互连，计算机网络间传送报文的方法
1986-1991年：NSFnet的子网从100个迅速增加到3000多个，成为Internet的基础
....
```

## IP地址与域名
```
IP地址：给处于网络中的每个计算机指定一个号码，一个唯一的地址，便于其它计算机识别。
IP地址是按照点分10进制的方式表示的，A.B.C.D

IP地址难以记忆，采样域名来进行表示，域名也是全球唯一的
一个IP地址可以绑定多个域名，一个域名同一时刻只能对应一个IP地址
```
